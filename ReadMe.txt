Steps to run the project:
1) Download the project on your local machine
2) Extract the project
3) Import the project as Maven project in IDE like Eclipse or IntelliJ
4) Go to src/test/java/take_home_webdriver_test.Webdriver_Tests
5) Run LoginTestCases.java as TestNG test.

Configuration:
1) Go to src/test/resources/application.properties
2) Change the browser value to chrome/firefox as per the requirement. By default firefox is selected.

Note: We are using latest geckodriver v0.29.1 & chromedriver 91.0.4472.19
      Application has been tested on Windows.
      TestCases successfully executed on chrome 92.0.4515.107 & firefox 90.0.2
